✅ 一、项目名称

AI销售获客Agent（ToB智能销售助手）


✅ 二、项目描述

项目解决的核心痛点：
当前ToB销售存在以下问题：

客户获取效率极低（大量无效拜访）

无法快速判断客户是否有真实需求

销售新人沟通能力弱，容易紧张、不知道说什么

客户信息碎片化，没有系统沉淀

跟进效率低，容易流失潜在客户


本项目通过AI Agent，实现“获客 + 判断 + 话术 + 跟进”全流程自动辅助。

✅ 三、核心逻辑


核心逻辑拆解：

多Agent协作系统（重点！）

系统由多个AI Agent组成：

客户分析Agent

输入：行业、地点

输出：潜在客户画像 + 是否值得拜访


话术生成Agent

根据行业生成：

开场白

破冰话术

加微信话术（真实用到的）



对话复盘Agent

输入：销售录音 or 对话记录

输出：

哪句话说得不好

哪里可以优化

推荐更优表达



跟进Agent

自动生成：

第二次跟进话术

微信消息模板

成交推进策略





---

长期记忆

系统会记录：

客户信息

历史沟通

意向评分


实现“越用越聪明”的销售AI


数据驱动优化

每天记录：

拜访数

成交率

拒绝原因


 AI自动分析：

> “你在哪类客户中成功率更高”

✅ 四、总结
> 本项目通过多Agent协作 + 长期记忆机制，将传统依赖经验的销售过程，转变为可量化、可优化的AI驱动流程。
在实际使用中，可帮助销售新人快速成长，提升转化率，并显著减少无效拜访时间。
